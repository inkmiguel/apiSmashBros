Super Smash Bros API

Una API REST para consultar información de personajes de Super Smash Bros.

Lista de personajes
GET /Personajes
Obtiene todos los personajes disponibles.
Detalle de personaje
GET /Personajes/<name>
Obtiene la información de un personaje específico por su name.
Consulta directa por nombre
http://localhost:3000/Personajes/
